# FutureShow — Landing Comparison

Three visual directions for the FutureShow landing page, side-by-side for consultation and decision.

**Live:** https://futureshow1.github.io/futureshow-compare/

## Versions

| Version | Status | Path |
|---|---|---|
| **V0 · Current** | Production status quo | `v0-current.html` |
| **V2 · Modernized** | Polished evolution | `v2/index.html` |
| **V3 · WIRED** | Magazine-style reinvention + 6 case studies | `v1-wired.html` |

## Structure

- `index.html` — the picker / comparison page (entry point)
- `v0-current.html` — copy of current production landing
- `v2/` — full futureshow-v2 build (modernized direction)
- `v1-wired.html` — WIRED magazine-style landing
- `projects/` — 6 case study pages for WIRED version
- `screens/` — hero screenshots + OG images
- `logos/` — client logos (HBO, MGM, KGHM, TVN, POLIN, Trans/Wizje, UW + typographic NIAiU, Teatr Dramatyczny)
- `og-template.html` — sharable OG image template

## How to use

1. Open https://futureshow1.github.io/futureshow-compare/
2. Click each version card to see the full landing
3. Each WIRED case study is its own page (`/projects/cardiosim.html` etc.)
4. Decide direction based on audience fit, not aesthetic preference alone
